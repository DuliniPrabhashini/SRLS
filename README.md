# Simple Register Login System (Spring Boot + MySQL)

This is a beginner-friendly full-stack web application built using **Spring Boot**, **Hibernate (JPA)**, **MySQL (XAMPP)**, and **HTML/CSS/JavaScript**.

The project demonstrates a simple **Register and Login system** without JWT authentication.

---

## 🚀 Features

- User Registration
- User Login
- MySQL database integration
- Hibernate ORM (Spring Data JPA)
- Simple frontend UI (HTML, CSS, JavaScript)
- Dashboard page after login
- DTO-based architecture
- REST API communication

---

## 🛠️ Technologies Used

### Backend:
- Java 17
- Spring Boot
- Spring Web
- Spring Data JPA (Hibernate)
- MySQL Driver

### Frontend:
- HTML
- CSS
- JavaScript (Fetch API)

### Database:
- MySQL (XAMPP)
